## 백엔드와 협업 내용

> 기획 방향

- 금융상품을 소개하고, 맞춤형 금융 상품 제공
- 금융상품 구상
  - 대출, 적금, 펀드, 카드, 멤버쉽
- 추천 상품
  - 사용자가 입력한 회원 정보를 바탕으로 금융 상품 추천

> 기능

- 회원가입
